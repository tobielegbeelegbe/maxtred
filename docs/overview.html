Overview

Entire app is wrapped by few providers for different purposes, such as authentication, passing routes, MUI theme & optional redux.

1. <Provider store={Store}> Redux store provider. This is optional if you don't want to use eCommerce, Scrumboard, Chat app.

2. <SettingsProvider> MatX uses context api to set application layout and color themes, for example we can set footer fixed, footer theme, set main theme & etc. Learn more about settings.

3. <MatxTheme> uses MUI's ThemeProvider and set application's main theme.

4. <AuthProvider> is used for user authentication. Learn more about authentication.
